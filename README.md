# Insurance Configuration

## Descripción

En este proyecto se agrupa la configuración del proyecto de Microservicios de Seguros, que incluye las siguientes configuraciones:

- `application.yml`: Configuración principal de la aplicación.
- `member-service.yml`: Configuración específica para el servicio de miembros.
- `policy-service.yml`: Configuración específica para el servicio de pólizas.
- `claim-service.yml`: Configuración específica para el servicio de reclamaciones.
- `gateway-service.yml`: Configuración específica para el servicio de gateway.
